Теория

Формула простых процентов

Используется, когда проценты не капитализируются:

S = P × (1 + r × n)

Где: S - итоговая сумма, P - первоначальная сумма, r - процентная ставка (в долях), n - срок в годах.

Формула сложных процентов

Используется, когда проценты капитализируются (прибавляются к основной сумме):

S = P × (1 + r)^n

Где: S - итоговая сумма, P - первоначальная сумма, r - процентная ставка (в долях), n - срок в годах.

Ежемесячная капитализация

Когда проценты начисляются каждый месяц:

S = P × (1 + r/12)^(12×n)

Где: r - годовая ставка, n - срок в годах.

Теория

Формула аннуитетного платежа

Ежемесячный платеж по аннуитетной схеме:

A = P × (r × (1 + r)^n) / ((1 + r)^n - 1)

Где: A - ежемесячный платеж, P - сумма кредита, r - месячная ставка (годовая ставка / 12), n - количество месяцев.

Общая сумма выплат

Общая сумма, которую заплатит заемщик:

S = A × n

Где: A - ежемесячный платеж, n - количество месяцев.

Теория

Формула дифференцированного платежа

Основная часть платежа (тело кредита):

Pосн = S / n

Процентная часть платежа:

P% = (S - Pосн × (k - 1)) × r

Где: S - сумма кредита, n - количество месяцев, r - месячная ставка, k - номер платежа.

Первый и последний платежи

Первый платеж:

P1 = S / n + S × r

Последний платеж:

Pn = S / n + (S / n) × r

Где: S - сумма кредита, n - количество месяцев, r - месячная ставка.

Теория

Формула будущей стоимости

Стоимость инвестиций через n лет:

FV = PV × (1 + r)^n

Где: FV - будущая стоимость, PV - текущая стоимость, r - процентная ставка, n - срок в годах.

Формула текущей стоимости

Сколько нужно инвестировать сейчас, чтобы получить нужную сумму в будущем:

PV = FV / (1 + r)^n

Где: FV - будущая стоимость, r - процентная ставка, n - срок в годах.

Регулярные инвестиции

Будущая стоимость регулярных инвестиций:

FV = PMT × [(1 + r)^n - 1] / r

Где: PMT - сумма регулярного платежа, r - процентная ставка, n - количество периодов.

Основные финансовые формулы

Вклады

Простые проценты

S = P × (1 + r × n)

Где: S - итоговая сумма, P - первоначальный вклад, r - процентная ставка (в долях), n - срок в годах.

Сложные проценты

S = P × (1 + r)n

Где проценты начисляются на сумму с предыдущими процентами.

Ежемесячная капитализация

S = P × (1 + r/12)12×n

Когда проценты начисляются каждый месяц.

Кредиты

Аннуитетный платеж

A = P × (r × (1 + r)n) / ((1 + r)n - 1)

Где: A - ежемесячный платеж, P - сумма кредита, r - месячная ставка, n - количество месяцев.

Дифференцированный платеж

Pk = P/n + (P - P×(k-1)/n) × r

Где: Pk - платеж в месяц k, P - сумма кредита, n - срок в месяцах, r - месячная ставка.

Общая переплата

Переплата = Сумма всех платежей - Сумма кредита

Показывает, сколько заемщик переплатит банку.

Инвестиции

Будущая стоимость

FV = PV × (1 + r)n

Где: FV - будущая стоимость, PV - текущая стоимость, r - ставка, n - срок.

Текущая стоимость

PV = FV / (1 + r)n

Сколько нужно инвестировать сейчас для получения нужной суммы в будущем.

Регулярные инвестиции

FV = PMT × [(1 + r)n - 1] / r

Где: PMT - сумма регулярного платежа, r - ставка за период, n - количество периодов.
